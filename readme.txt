== Description ==

= PERFORMANT =

The Plugin only loads the Fonts you selecte and makes shure you site stay fast.

We have build a native ui to select your Google Fonts.
With a fancy preview of every font and an easy way to add and remove fonts with just one klick.

TK Google Fonts will automaticly add selected fonts to your site and you can than use this fonts in your css without any extra step needed.

== Installation ==

= Minimum Requirements =

* WordPress 3.5 or greater
* PHP version 5.2.4 or greater
* MySQL version 5.0 or greater

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don’t even need to leave your web browser. To do an automatic install of TK Google Fonts , log in to your WordPress admin panel, navigate to the Plugins menu and click Add New.

In the search field type “TK GOOGLE FONTS ” and click Search Plugins. Once you’ve found the plugin you can view details about it such as the the point release, rating and description. Most importantly of course, you can install it by simply clicking Install Now. After clicking that link you will be asked if you’re sure you want to install the plugin. Click yes and WordPress will automatically complete the installation.

= Manual installation =

The manual installation method involves downloading our eCommerce plugin and uploading it to your webserver via your favourite FTP application.

1. Download the plugin file to your computer and unzip it
2. Using an FTP program, or your hosting control panel, upload the unzipped plugin folder to your WordPress installation’s wp-content/plugins/ directory.
3. Activate the plugin from the Plugins menu within the WordPress admin.

= Upgrading =

Automatic updates should work like a charm; as always though, ensure you backup your site just in case.

Yes; TK Google Fonts will work with any theme, you can use the fonts on your css

but if your them has theme options for your theme fonts, the theme needs to support TK Google Fonts.

or you build a smal function to add your fonts to your fonts options.
